Dental Clinic

2024

A static website for a dental clinic, with a clean and modern design. The website is built with React, Next.js 14 App Router, TypeScript, and Tailwind.

Features

  • Mapbox integrattion

  • Emialjs for sending messages

  • Form validation with zod

  • Conditional classes with clsx

  • Designed with Tailwind

  • Modern UI on top of shadcn/ui

  • Responsive design

Note Taking App

2023-present

Build a fullstack Keep Clone with React, Next.js 14 App Router, TypeScript, Firebase, and Tailwind.

Features

  • Real-time database with Firebase

  • Create, update, and delete notes

  • Light and Dark mode with next-themes

  • Auth with Google

  • Auth with Github

  • Search notes

  • Pin notes

  • Archived notes

  • Change background color of the note

  • Image upload

  • Designed with Tailwind and NextUI

  • Responsive design
